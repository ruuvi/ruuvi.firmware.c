◆ app_led_init()

rd_status_t app_led_init ( void  )

Initialize leds.

After calling this function all LED GPIOs are high-drive outputs and in inactive state. Led active states and pins are loaded from board definition.

Return values
RD_SUCCESSon successful init.
RD_ERROR_INVALID_STATEif already initialized.
